String Making

In-Range Archery can custom make recurve strings to your specific requirements including colour, material, strand count, serving type and nock fit.

String specifications can make a huge difference to the performance of your shooting.

MATERIAL TYPES

  • Dacron: is a good string material for beginnerâ€™s basic bows but is a material that keeps on stretching over time.
  • Fast Flight material is much better spec, far less stretch, thinner material but still forgiving.
  • BCY 8125 material has very little stretch, thinner and significantly more performance.

NUMBER OF STRANDS

The number of strands can affect string performance. More strands mean a more stable shot but slower which can have the effect on the behaviour of the arrow, likewise if you have fewer strands it can increase arrow speed slightly and weaken arrow spine but is also less forgiving.

SERVING TYPE

  • A simple braided BCY 400 serving works fine and is the serving you would find on most mass-produced factory strings.
  • Halo serving is more expensive but performs well, slips off the tab smoothly and quickly. Halo materials come in various thicknesses to allow for a perfect nock fit.
